# Telecom SMS Traffic Analysis

## Project Overview

Developed a Telecom SMS Traffic Analysis project using Power BI, SQL, and Excel to analyze SMS traffic performance, delivery trends, operator activity, and message distribution patterns. The project focuses on monitoring telecom messaging data to identify traffic trends, delivery performance, and operational insights.

---

## Objectives

- Analyze SMS traffic and delivery performance
- Monitor operator-wise message activity
- Identify delivery failure trends
- Analyzed region & industry-wise SMS traffic
- Generate business insights from telecom messaging data

---

## Features

- SMS Traffic Volume Analysis
- Operator-wise Performance Tracking
- Delivery Success & Failure Analysis
- Message Type Distribution Analysis
- Region & industry-wise SMS traffic Analyis
- Interactive Dashboards and Visualizations

---

## Tools Used

- Power BI
- SQL
- Excel

---

## Key Insights

- Identified peak SMS traffic periods
- Analyzed operator-wise message performance
- Compared delivery success and failure trends
- Analyzed region & industry-wise SMS traffic 
- Improved telecom traffic visibility using data visualization
